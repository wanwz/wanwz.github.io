在此罗列一些vscode的常用快捷键。
<!-- more -->

## 注释

```
单行注释(也可用于多行注释，但模式是单行注释)：ctrl + /
多行注释：alt + shift + a
取消注释：ctrl +k +u 或 重复上面注释的操作
```

## 位置

```
Home 光标跳转到行头

End 光标跳转到行尾

Ctrl + Home 跳转到页头

Ctrl + End 跳转到页尾
```

## 调试快捷键

```
F5 启动调试
F9 添加解除断点
```
